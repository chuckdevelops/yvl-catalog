Why download button not work?

Download file should start automatically after clicking on “Download now” button. If the button does not work, our servers are very overloaded. Click the button only once. After clicking you must wait until the download will start. High loads are often temporary, so you can try again after a few minutes.

Why audio/video player not display?

Our file hosting supports the official HTML5 audio/video extension. Player should be automatically generate for file like: .mp3 .m4a .wav .ogg .m4a .mp4 .webm .mkv .m4v

Why audio file (mp3) not display image cover?

Cover image for mp3 files are generated from audio tags. Your mp3 must contain the image in its source.

Why browser alert about viruses when try to download?

Uploaded files are in most cases by anonymous users. If you want to download a file make sure it’s safe for you. We don’t check files to find viruses. Sometimes browsers make mistakes and block normal files without malware. You can disable scanning downloading files in your operating system firewall.

Why download files is interrupting after few minutes?

In most cases this is due to an interrupted internet connection to you. Next case when our servers are very overloaded, you need to try again in a few minutes.
